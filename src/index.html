Welcome

Minimal Boilerplate for building content for hicetnunc.xyz based on Laravel Mix 6, TailwindCSS 2, PostCSS 8, and ES6 Vanilla JS components.

The file you're looking at is the index.html file in the repository you just cloned.

Edit that and the supporting files, while running yarn watch from your command line. When you're ready to deploy, run yarn production and ZIP up the dist folder. Upload that to Hic Et Nunc for minting and be careful to check the preview.
